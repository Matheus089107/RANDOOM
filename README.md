# 💀 Doom Standalone Project

Este é um repositório independente para o projeto **Doom**, contendo tanto a versão original em Python quanto a nova evolução de alta performance em C++.

## 📂 Estrutura do Repositório

-   `main.py`: Versão original em **Python** (2.5D Raycaster com Multijogador).
-   `cpp_version/`: Nova versão em **C++** usando a biblioteca **Raylib**.
-   `images/`: Assets visuais (texturas, sprites, animações).
-   `server.py`: Servidor de relay para o modo multijogador.

## 🚀 Como Executar

### Versão Python
Certifique-se de ter o Pygame instalado:
```bash
pip install pygame
python main.py
```

### Versão C++
Veja as instruções detalhadas em `cpp_version/setup_guide.md`.
